# 🧠 NeuroScan AI - Brain Tumor Detection System

An AI-powered Brain Tumor Detection System built using CNN, TensorFlow, and Flask. The application analyzes MRI brain scan images and predicts whether a tumor is present.

## 🚀 Features

- MRI Image Upload
- Brain Tumor Detection using CNN
- Real-time Prediction
- Confidence Score Display
- Flask Web Interface

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- Flask
- NumPy
- OpenCV
- HTML & CSS

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Harini-sri-r/NeuroScan-AI-Brain-Tumor-Detection-System.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```

4. Open in browser

```
http://127.0.0.1:5000
```
